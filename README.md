# Financial & Geopolitical AI Agent (LangChain ReAct)

A simple full-stack AI agent built using **Node.js + LangChain (ReAct Agent)**.

## 🚀 Features
- Latest geopolitical news (GNews API)
- Mock database (store/delete news)
- Calculator
- Gold price info
- LLM-powered responses

## 🛠️ Tech Stack
- Node.js, Express
- LangChain + OpenAI
- HTML, JS frontend

## 📂 Structure
langchain_sdk_test/
├── backend/
├── frontend/

## ⚙️ Setup
1. Clone repo
git clone <your-repo-url>

2. Install dependencies
cd backend
npm install

3. Create `.env` in backend
OPENAI_API_KEY=your_key  
GNEWS_API_KEY=your_key  

4. Run server
node server.js

5. Open in browser
http://localhost:3000

## 🧪 Example Queries
- latest news
- gold price
- 25 * 10
- delete news
